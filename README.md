# Tetris
## Controls

 * A : Rotate Left
 * S : Rotate Right
 * Left : Move Left
 * Right : Move Right
 * Down : Downward Acceleration
 * Space : Pause

## Debug Controls

### Gameplay Controls

 * Q : Regular Mode
 * W : Frozen Mode
 * E : Infinity Mode
 * - : Slow Down
 * = : Speed Up

### Tetromino Switching

 * 1 : I Tetromino
 * 2 : O Tetromino
 * 3 : T Tetromino
 * 4 : J Tetromino
 * 5 : L Tetromino
 * 6 : S Tetromino
 * 7 : Z Tetromino
 * 8 : Debug Tetromino
